# .lett

My dotfiles.

## Requirements

### On Arch

* [hyprland](https://hypr.land/): Hyprland, Waybar etc are not part of the installation script, and installing this might break your setup right now (until I fix it)
* zsh: This considers ZSH is your default sheel

### On mac:

* [brew](https://brew.sh/)

## Automatic installation:

** Pleave review the script before running it, it might break your entire setup **

```
curl -L https://raw.githubusercontent.com/gabriellett/.lett/master/install.sh | sh
```

## Manual installation

Then clone this repo in your home folder:
	
```
git clone https://github.com/gabriellett/.lett ~/.lett
```

And check install.sh on details
